开发一个运行在 macOS 上的桌面终端宠物。

主要功能：

* 宠物以透明悬浮窗口显示在桌面，可拖动、置顶并记住位置
* 支持待机、开心、震惊、工作、睡觉等动画状态
* 通过 zsh Shell Hook 感知终端命令开始、成功、失败和运行时长
* 命令成功时庆祝，命令失败时吐槽或给出简单排查建议
* 连续报错时提醒用户不要重复执行同一命令
* 长时间运行命令时显示等待状态
* 监控 CPU、内存、电池和磁盘空间，并在异常时提醒
* 用户长时间无操作时自动睡觉或提醒休息
* 点击宠物可以聊天、查看系统状态和最近终端事件
* 提供温柔、毒舌、技术型三种性格
* 支持自定义 OpenAI 兼容 API 或本地 Ollama
* AI 用于解释报错和生成宠物回复，不自动执行命令
* 提供菜单栏图标，可暂停监听、隐藏宠物、打开设置和退出
* 支持开机启动、免打扰、发言频率和宠物大小设置
* 默认保护隐私，不记录密码、Token 和完整终端输出
